Transformación (Text-transform:)

Transformation uppercase

La propiedad text-transform en CSS se utiliza para controlar la capitalización del texto de un elemento. Uno de los valores más comunes para esta propiedad es uppercase, que convierte todo el texto del elemento seleccionado a mayúsculas.

Transformation lowercase

La propiedad text-transform en CSS también puede utilizarse para convertir todo el texto de un elemento a minúsculas utilizando el valor lowercase. Esto es útil para asegurarse de que todo el texto se muestre en letras minúsculas.

Transformation capitalize

La propiedad text-transform en CSS puede utilizarse para convertir la primera letra de cada palabra a mayúscula utilizando el valor capitalize. Esto es útil cuando deseas que el texto se muestre en formato de título o capitalizado.

Texto y sus estilos

En CSS, los estilos de texto se utilizan para controlar la apariencia del texto en una página web. Esto incluye propiedades para la fuente, tamaño, color, alineación, espaciado y otros efectos.

  1. Color: Define el color del texto.
  2. Font-family: Especifica la familia de fuentes para el texto. Puede incluir varias fuentes como una lista de respaldo.
  3. Font-size: Define el tamaño del texto. Puede especificarse en unidades como píxeles (px), em (em), rem (rem), porcentajes (%), etc.
  4. Font-weight: Define el grosor del texto. Los valores comunes incluyen normal, bold, y números de 100 a 900.
  5. Font-style: Define el estilo de la fuente. Los valores comunes son normal, italic, y oblique.
  6. Text-align: Define la alineación del texto. Los valores comunes son left, right, center, y justify.
  7. Text-decoration: Define las decoraciones del texto, como subrayado, sobrelineado y tachado. Los valores comunes son none, underline, overline, y line-through.
  8. Line-height: Define la altura de la línea del texto, afectando el espaciado entre las líneas de texto.
  9. Letter-spacing: Define el espaciado entre los caracteres del texto.
  10. Word-spacing: Define el espaciado entre las palabras del texto.
  11. Text-transform: Controla la capitalización del texto. Los valores comunes son none, capitalize, uppercase, y lowercase.
  12. Text-shadow: Aplica una sombra al texto. Se pueden especificar valores para el desplazamiento horizontal y vertical, el desenfoque y el color.
  13. White-space: Controla cómo se maneja el espacio en blanco dentro del texto. Los valores comunes son normal, nowrap, pre, pre-wrap, y pre-line.

Alineamiento (Text-align:)

Alignment Left

La propiedad text-align en CSS se utiliza para controlar la alineación del texto dentro de un elemento. Uno de los valores más comunes es left, que alinea el texto hacia la izquierda del contenedor.

Alignment Right

La propiedad text-align en CSS se utiliza para controlar la alineación del texto dentro de un elemento. Uno de los valores más comunes es right, que alinea el texto hacia la derecha del contenedor.

Alignment Center

La propiedad text-align en CSS se utiliza para controlar la alineación del texto dentro de un elemento. Uno de los valores más comunes es center, que centra el texto horizontalmente dentro del contenedor.

Alignment Justify

La propiedad text-align en CSS se utiliza para controlar la alineación del texto dentro de un elemento. El valor justify justifica el texto, distribuyendo el espacio entre las palabras para que el texto ocupe todo el ancho del contenedor, lo que crea un borde recto en ambos lados.

Subrayado (Text-decoration:)

Decoration overline

La propiedad text-decoration en CSS se utiliza para aplicar decoraciones al texto, como subrayado, tachado, o una línea encima del texto. El valor overline añade una línea por encima del texto.

Decoration line-through

La propiedad text-decoration en CSS se utiliza para aplicar decoraciones al texto, como subrayado, tachado, o una línea encima del texto. El valor line-through añade una línea a través del texto, lo que se conoce comúnmente como tachado.

Decoration underline

La propiedad text-decoration en CSS se utiliza para aplicar decoraciones al texto, como subrayado, tachado o una línea encima del texto. El valor underline añade una línea debajo del texto, lo que se conoce comúnmente como subrayado.

Decoration overline-underline

La propiedad text-decoration en CSS permite aplicar múltiples decoraciones al texto, como subrayado, una línea encima del texto, o ambas. Para añadir tanto una línea debajo del texto como una línea encima del texto, puedes combinar los valores overline y underline.

Fuentes (Font-family:)

Font type Serif

Las fuentes Serif son un tipo de tipografía caracterizada por tener pequeños adornos o líneas en los extremos de los caracteres. Estas decoraciones, conocidas como "serifas", proporcionan un aspecto clásico y formal al texto. Las fuentes Serif se utilizan comúnmente en impresos, libros y publicaciones académicas.

Font type Sans-Serif

Las fuentes Sans-Serif son un tipo de tipografía que no tiene las pequeñas decoraciones o líneas en los extremos de los caracteres que se encuentran en las fuentes Serif. "Sans" significa "sin" en francés, por lo que "Sans-Serif" literalmente significa "sin serifas". Estas fuentes suelen tener un aspecto limpio y moderno, y son comúnmente utilizadas en diseño web y interfaces digitales debido a su legibilidad en pantallas.

Font type Monospace

Las fuentes Monospace son un tipo de tipografía donde cada carácter ocupa el mismo ancho horizontalmente, lo que significa que todos los caracteres, incluyendo letras, números y signos de puntuación, ocupan el mismo espacio en la línea. Este tipo de fuente es útil en situaciones donde la alineación precisa de texto es importante, como en programación, tablas y documentos donde se necesita una estructura de columna clara y uniforme.

Font type Cursive

Las fuentes Cursive en CSS se refieren a tipografías diseñadas para imitar la apariencia de la escritura a mano cursiva o manuscrita. Estas fuentes suelen tener conexiones entre las letras y un flujo que imita el trazo de un bolígrafo o un rotulador. Se utilizan principalmente para añadir un toque personal o artístico a los textos en aplicaciones web y diseños creativos.

Font type Fantasy

Las fuentes Fantasy en CSS se refieren a tipografías decorativas y no convencionales que suelen tener diseños extravagantes, creativos o temáticos. Estas fuentes están diseñadas para llamar la atención y agregar un toque distintivo a los textos en diseños gráficos y páginas web donde se busca un estilo único y visualmente impactante.
